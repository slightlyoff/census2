The "Guide Me" mode walks you through benchmarks for various methods of loading data in RIAs. This includes descriptions of each method and explanations for the results. The "Navigator" mode allows you to run the benchmarks on your own. To continue in "Guide Me" mode click the next button above, or select the benchmark from the dropdown.

Note: You can see the output of the last benchmark by selecting the "Output" panel on the right. To see all the timings that go into the running averages displayed by the charts, select the "Data" tab on the right.
